# Render Blueprint path test

Minimal repo to test whether the **Deploy to Render** button prefills **Blueprint Path** or if you must type `render.yaml` manually.

- Single file: `render.yaml` at repo root (one free static site).
- Public repo.

## Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/aculich/render-blueprint-test/tree/main)

When the Blueprint form loads, check **Blueprint Path**. If it is empty or a placeholder, type `render.yaml` and continue.
